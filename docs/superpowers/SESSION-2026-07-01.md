### Bugs achados e corrigidos nessa sessão
- **ThinkingBlock**: `claude-sonnet-5` liga extended thinking por padrão → resposta vem com
  `ThinkingBlock` no `content[0]`; código lia `content[0].text` → crash silencioso. Fix:
  varre blocos e pega o primeiro com `.text`. (Afeta todo `depth="deep"`, inclui `/plano`.)
- **Resposta longa** (>4096) estourava `reply_text` fora do try → thread órfã. Fix: chunk.
- **`_authorized`** removido de `on_ask_button` por um implementer → restaurado.
- **Estilo**: resposta longa/pouco objetiva + markdown cru + `finalizar` apagava a resposta
  → `extra_system` conciso/sem-MD + `edit_message_reply_markup(None)` preserva o texto.
